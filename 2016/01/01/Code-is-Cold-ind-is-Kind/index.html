Code is Cold, Mind is Kind

发表于 2016-01-01 | | 浏览 次

  只是自己随手写写的地方。


  要说玩Linux的爱豆是谁，肯定是我大Linus啊！大神在上！

  Dock，毕业于国内某普通211校，学的是自动化，但是自动控制理论、电机控制没学会多少，走上了挨踢这条路。从8位的51、16位的MSP、32位的STM32一直搞到了现在的嵌入式Linux，Zynq，最近又莫名其妙的玩上了TensorFlow。爱折腾，没办法。陌上花开，可缓缓归矣

  回望大学、回望初进职场，仿佛昨日。想当初初入校园，整日晕头转向，不知路在何方…….

  • 想当初，从报名参加某单片机竞赛，从不知单片机为何物，到在5个月内学会C、51、使用比赛用的片子做出一个可穿戴的作品，更奇怪的是还拿了个一等奖，也许那是我大学四年最光彩的时刻了。
  • 大二破格进入了学校的飞思卡尔智能车队,把能挤出的时间都投进去了，但是还是一个安慰奖。大三再战一年，到最后因为备件的问题……..，从未后悔，因为拿到的不应该是那一个名次，还有那些好朋友，刘X，张X。。。。。
  • 参加国内某新杯，带几个学霸，帮他们在保研加分上有那么一点助力。。。。。。。
  • 进入NXP半导体实习，也许是我参加飞思卡尔锲而不舍的精神感动了他们。
  • 当前最好的收获还是收获了一个爱自己的女票。”XX你没我玩的溜”，“我有女票”; “XX我的电脑性能比你好”，”我有女票“……无限不循环循环中…….，“XX我女朋友比你的漂亮”，”我承认，新垣结衣可不是你一个人的女朋友”，(￣▽￣)／
  • 现在在北京发展，在嵌入式Linux上的路越奔越远………

  如果你也曾像我一样，希望你确认好自己的方向，管他什么阻碍，科技技术日新月异，一定不要犯懒，学好新技术，下面是Dock在大学期间学过的一些MOOC课程，谁能想到现在Python突然火了。。。。

做好技术储备，也许这是我的第一个建议吧。。。

未完待续中。。。。。。
